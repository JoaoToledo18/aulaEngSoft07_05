# Requisitos Funcionais

## RF01 — Gerenciar cinemas

O sistema deve permitir o cadastro, edição, consulta e remoção de cinemas da rede.

---

## RF02 — Gerenciar filmes

O sistema deve permitir o cadastro, edição, consulta e remoção de filmes.

---

## RF03 — Gerenciar sessões

O sistema deve permitir o cadastro, edição, consulta e remoção de sessões de filmes.

---

## RF04 — Associar filmes aos cinemas

O sistema deve permitir associar filmes aos cinemas por meio das sessões.

---

## RF05 — Registrar público das sessões

O sistema deve permitir registrar a quantidade de espectadores presentes em cada sessão.

---

## RF06 — Consultar programação

O sistema deve permitir consultar os filmes e sessões disponíveis em cada cinema.

---

## RF07 — Consultar informações dos filmes

O sistema deve permitir consultar informações dos filmes, como:

- gênero;
- duração;
- diretor;
- elenco;
- classificação indicativa.

---

## RF08 — Consultar total de público

O sistema deve permitir consultar o total de público:

- por sessão;
- por filme;
- por cinema.
